# Homebrew Tap for niha CLI

Install the niha CLI:

```bash
brew install --cask niha-and-co/tap/niha
```

Upgrade:

```bash
brew upgrade --cask niha
```

Alternative install (no Homebrew):

```bash
curl -fsSL https://cli.nihaandco.com/install.sh | bash
```

## About

niha is an AI-native governance CLI for development teams. Learn more at [nihaandco.com](https://nihaandco.com).
